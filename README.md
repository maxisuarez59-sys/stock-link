# Stock-Link

## Descripción

Aplicación web para la gestión automatizada de pedidos a proveedores vía WhatsApp para el sector de hostelería.

## Tecnologías

- **Backend:** PHP (XAMPP)
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Base de Datos:** MySQL
- **Servidor:** Apache
